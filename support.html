Videomera Support

Need help with importing, queueing, or exporting? Email eraappai@gmail.com.

Helpful details to include

  • Your system version.
  • Whether the files are on your local device, an external drive, or a network/cloud-synced folder.
  • The source format, such as MP4 or MOV, and whether the clips came from the same recording system.
  • The exact export status or error message shown by Videomera.

Common fixes

  • If Videomera cannot open a file, add the folder again using the in-app folder picker so the system grants access.
  • If there is not enough disk space, export to the external drive that contains the source videos or another drive with enough free space.
  • Fast-copy merge is available when clips share compatible video settings. Mixed formats may need the safer compatibility export path.
  • If a synced folder is slow, export locally first, then let your chosen sync service upload the finished file.
Videomera creates a new merged export and leaves your source clips in place.
